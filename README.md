# Librarian Portfolio site
A portfolio site for a local librarian. Made with WordPress and PHP.

## Plugins
* WordFence
* UpdraftPlus
* Smush
* Central Color Palette
* Disable Comments
* Simple Social Icons
* Titan Anti-spam and Security
* Yoast SEO
